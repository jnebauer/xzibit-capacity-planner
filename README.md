# Capacity Planner (React + Vite + TypeScript) — Production-Ready

This project embeds your initial datasets (115 jobs, employees, curves), and uses Recharts for graphs.

## Run locally
npm install
npm run dev   # http://localhost:5173

## Build for production
npm run build
npm run preview

## Deploy to Vercel
1) Push this folder to a GitHub repository.
2) In Vercel: **New Project** → Import your repo.
3) Framework: **Vite**. Build command: `npm run build`. Output directory: `dist`.
4) Deploy. You'll get a permanent URL you can share with your team.
